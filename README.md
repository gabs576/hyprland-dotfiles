# hyprland-dotfiles

This is my first ever hyprland rice I have done, it uses waybar as main status bar, rofi as application launcher, swaync as notifications daemon (if you want just change on ~/.config/hypr/hyprland.conf at exec-once to dunst)
and uses matugen as color generator

# commands

To open the web browser it uses SUPER+B;
For file manager SUPER+E;
For rofi SUPER+Space;

# Rofi

On rofi it has some scripts it can help you during your daily drive

For wallpaper selector is SUPER+W;
For wifi is SUPER+T;
Bluetooth is SUPER+M;
For some notes is SUPER+N;
For power options is SUPER+P;

# IDE

It uses neovim with Lazy.nvim, as lsp-config, autocompletion, debuggin (on the moment only for go language), neo-tree, telescope, treesitter, lualine and none-ls

# commands neovim

Ctrl+n to use neo-tree and Space+n to close
Ctrl+p to find files
Space+gf to format files
Space+fg for live grep files and folder
Space+dt to make a brakepoint
Space+dc to start debbug the line marked as brakepoint
